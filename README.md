# m7r-web-components

This repository includes components for the mentalizr web frontend.

## Status

Under development.

## Prepare project

    npm install
    
## Build project

    ./node_modules/.bin/webpack
    
or

    gradle webpack
    
## See demo

    ./node_modules/.bin/webpack-dev-server
    
Call `http://127.0.0.1:8080/html/demo-mc-question.html` in your browser.
